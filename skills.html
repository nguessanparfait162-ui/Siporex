CSS
Intermédiaire

Le CSS (Cascading Style Sheets) me permet de donner vie visuellement aux pages web. Je maîtrise les propriétés de mise en page modernes comme Flexbox et CSS Grid pour créer des interfaces élégantes et adaptées à tous les écrans (responsive design). J'utilise les variables CSS, les animations et les transitions pour créer des expériences utilisateur fluides et professionnelles, tout en respectant les principes d'un design cohérent et moderne.
